# SmsBark - 短信验证码转发工具

SmsBark 是一个轻量级的 Android 应用，用于自动捕获手机接收到的短信验证码，并通过 Bark 协议转发到您的 iOS 设备。

## 核心功能

*   **自动监听**: 实时监听接收到的短信。
*   **智能识别**: 自动识别包含“验证码”或“code”关键词的短信。
*   **即时转发**: 通过 HTTP 请求将短信内容推送到 Bark 服务器。
*   **状态统计**: 每日实时展示转发成功和失败的条数。
*   **后台保活**: 内置权限检测，指导用户开启必要的后台权限。

## 使用指南

### 1. 配置 Bark
1. 在 iOS 设备上安装 Bark App。
2. 获取您的服务器地址（通常为 `https://api.day.app`）和 Token。
3. 打开 SmsBark，点击右上角设置，填入上述信息并保存。

### 2. 权限设置 (至关重要)
为了保证 App 能在后台长期稳定运行，请务必完成以下设置：

*   **授予短信权限**: App 首次启动或在首页点击警告卡片，授予“读取短信”权限。
*   **忽略电池优化**: 在首页点击警告卡片，允许 App 在后台不受限制地运行。
*   **允许自启动 (重要)**:
    *   请手动前往手机系统的 **设置 -> 应用管理 -> SmsBark -> 权限/自启动管理**。
    *   开启 **“允许自启动”**、**“允许后台活动”** 或类似选项。
    *   不同品牌手机（小米、华为、OPPO、Vivo等）设置路径略有不同，请务必开启，否则 App 进程被杀后将无法转发。

### 3. 锁定后台
建议在手机的多任务界面（最近任务列表）中，将 SmsBark **锁定**（通常是下拉卡片或点击锁图标），防止一键清理时被误杀。

## 隐私说明
SmsBark 所有的转发行为均直接发生在您的手机与 Bark 服务器之间。App 不会收集、存储或上传您的任何短信内容到第三方服务器。
